Política de Privacidade — PagueMenos ML

Última atualização: 30 de junho de 2025

Esta política de privacidade descreve como o aplicativo PagueMenos ML, desenvolvido por Marcos Dunker, lida com informações dos usuários.

1. Coleta de dados

O PagueMenos ML não coleta, armazena nem compartilha dados pessoais dos usuários. O aplicativo funciona como um buscador de produtos que exibe resultados do Mercado Livre e redireciona o usuário para o site oficial.

2. Dados de uso

Não rastreamos o comportamento individual dos usuários dentro do app. Nenhuma informação de identificação pessoal é coletada ou transmitida.

3. Redirecionamento para terceiros

Ao clicar em um produto, o usuário é redirecionado para o site do Mercado Livre (mercadolivre.com.br). A partir desse momento, a política de privacidade do Mercado Livre se aplica.

4. Links de afiliado

Os links de produtos contêm parâmetros de afiliado do Mercado Livre, o que nos permite receber uma comissão quando uma compra é realizada — sem nenhum custo adicional para o usuário.

5. Segurança

Toda a comunicação entre o app e nosso servidor é realizada por meio de HTTPS com certificado SSL.

6. Crianças

Este app não é direcionado a crianças menores de 13 anos e não coleta conscientemente informações de menores de idade.

7. Alterações nesta política

Podemos atualizar esta política ocasionalmente. Notificaremos sobre mudanças significativas por meio do próprio aplicativo.
